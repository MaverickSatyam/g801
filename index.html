Future Value (FV) Formula

The Future Value of an investment is calculated as:

$$FV = PV (1 + r)^n$$
  • PV: Present Value (The initial amount)
  • r: Interest rate per period
  • n: Number of periods

Understanding this is crucial for long-term investment planning.

Walk through each variable in the formula. Use a simple example: $100 invested for 1 year at 5% interest is $105. Point out how the list items use fragments.

Python Implementation of FV

Translating the formula into a simple Python function:


def calculate_fv(pv, rate, periods):
    # This function calculates the Future Value
    fv = pv * (1 + rate)**periods
    return fv

# Example usage:
initial_investment = 1000
annual_rate = 0.05
years = 5

result = calculate_fv(initial_investment, annual_rate, years)

# The result is: 1276.28
